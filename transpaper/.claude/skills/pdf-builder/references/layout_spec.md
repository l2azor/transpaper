# PDF 레이아웃 사양

## 용지 및 여백
| 항목 | 값 |
|------|-----|
| 용지 크기 | A4 (210 × 297 mm) |
| 상단 여백 | 25 mm |
| 하단 여백 | 25 mm |
| 좌측 여백 | 25 mm |
| 우측 여백 | 25 mm |
| 텍스트 영역 | 160 × 247 mm |

## 폰트 크기 및 줄간격

| 요소 | 폰트 크기 | 줄간격 (leading) | 정렬 |
|------|-----------|-----------------|------|
| 논문 제목 | 16pt | 22pt | 가운데 |
| 저자명 | 10pt | 14pt | 가운데 |
| 초록 라벨 | 11pt | 15pt | 가운데 |
| 초록 본문 | 9pt | 14pt | 양쪽 정렬 |
| 키워드 | 9pt | 13pt | 왼쪽 정렬 |
| H1 (섹션 제목) | 13pt | 18pt | 왼쪽 정렬 |
| H2 (하위 섹션) | 11pt | 16pt | 왼쪽 정렬 |
| H3 (하위하위 섹션) | 10pt | 14pt | 왼쪽 정렬 |
| 본문 | 10pt | 16pt | 양쪽 정렬 |
| 캡션 (그림/표) | 9pt | 13pt | 가운데 |
| 참고문헌 | 8pt | 12pt | 왼쪽 정렬 |
| 각주 | 8pt | 11pt | 왼쪽 정렬 |

## 간격 규칙
- 제목 아래: 6mm
- 섹션 제목(H1) 위: 8mm, 아래: 3mm
- 하위 제목(H2) 위: 5mm, 아래: 2mm
- 본문 문단 간: 2mm
- 초록 좌우 여백: 15mm (본문보다 들여쓰기)
- 본문 첫 줄 들여쓰기: 5mm
- 참고문헌 hanging indent: 5mm

## 캡션 규칙
- 그림 캡션: `[그림 N. 캡션 텍스트]` 형식으로 대괄호 포함
- 표 캡션: `[표 N. 캡션 텍스트]` 형식으로 대괄호 포함
- Phase 1에서는 이미지 삽입 없이 캡션만 표시

## 참고문헌 규칙
- 번호가 있는 항목은 hanging indent로 정렬
- 원문 그대로 유지 (번역하지 않음)
