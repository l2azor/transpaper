# Translation Glossary (전역 용어집)

이 용어집은 논문 번역 시 일관성을 위해 사용됩니다.
새로운 논문을 번역할 때마다 여기에 등록된 용어를 우선 참조합니다.

## 사용법
- `영문 용어` → `한국어 번역 (영문 원어)` 형식으로 기록
- 고유명사, 이론명, 방법론명은 원어 그대로 유지
